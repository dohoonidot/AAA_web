# 휴가 AI 작성 메뉴얼

> AI를 활용한 휴가 신청 및 관리 기능 안내

---

## 1. 휴가상신

### 개요
사내업무 AI 채팅에서 날짜, 연차종류, 반차여부, 사유를 입력하여 휴가 상신을 요청할 수 있습니다.

### 사용 방법
채팅창에 자연어로 휴가 신청 요청을 입력합니다.

### 예시
- `"내일 휴가써줘"`
- `"3월4일 휴가써줘 반차로"`
- `"다음주 화요일 오후반차써줘"`
- `"이번주 금요일 연차 신청해줘 병원 예약때문에"`

### 입력 항목
| 항목 | 필수 여부 | 설명 |
|------|----------|------|
| 날짜 | 필수 | 휴가 시작일 (종료일 미입력시 당일) |
| 연차종류 | 선택 | 미입력시 기본 연차 |
| 반차여부 | 선택 | 오전반차, 오후반차 지정 가능 |
| 사유 | 선택 | 미입력시 "개인사유"로 자동 입력 |

### 직접 작성 방법
AI를 사용하지 않고 직접 작성하려면:
1. 사이드바 메뉴에서 **휴가관리** 클릭
2. **"휴가 작성"** 버튼 클릭
3. 휴가 신청 폼 작성 후 제출

---

## 2. 휴가 부여 상신

### 개요
예비군, 민방위, 결혼, 경조사 등 특별휴가 부여가 필요한 경우 AI를 통해 자동으로 전자결재를 진행할 수 있습니다.

### 사용 방법 (예비군 휴가 부여 예시)

1. **서류 첨부**: 예비군 관련 서류를 채팅에 첨부
2. **요청 입력**: `"예비군 휴가 부여좀 해줘"` 입력
3. **자동 처리 프로세스**:
   - 휴가 부여 상신 전자결재 폼 자동 작성
   - 관리자(경영관리실) 승인 대기
   - 승인 후 AAA에서 해당 예비군 연차 상신 폼 자동 작성
   - 수정 필요시 수정 후 상신

### 지원되는 특별휴가 종류
- 예비군 훈련
- 민방위 훈련
- 결혼 휴가
- 경조사 휴가
- 기타 회사 규정에 따른 특별휴가

### 직접 작성 방법
처음부터 직접 작성하고 싶은 경우:
1. 채팅방 상단 메뉴 클릭
2. **"전자결재 상신 초안"** 선택
3. 해당 폼에서 직접 작성

---

## 3. 전자결재 승인자/참조자 저장

### 개요
전자결재 시 자주 사용하는 승인자와 참조자를 저장하여 빠르게 선택할 수 있습니다.

### 주의사항
- 전자결재용 승인자/참조자는 **휴가 상신용과 별도로 저장**됩니다
- 두 기능은 서로 연관이 없으며, 각각 독립적으로 관리됩니다

### 저장 위치
| 구분 | 저장 위치 | 용도 |
|------|----------|------|
| 휴가 상신용 | 휴가 신청 폼 | 휴가 결재 |
| 전자결재용 | 전자결재 폼 | 일반 전자결재 |

---

## 참고 사항

### AI 휴가 신청 팁
1. **명확한 날짜 표현**: "내일", "다음주 월요일", "3월 15일" 등 명확하게
2. **반차 지정**: "오전반차", "오후반차"로 명시
3. **사유 입력**: 결재 승인율을 높이려면 구체적인 사유 입력 권장

### 문의
- 시스템 오류 발생 시: IT 담당자 문의
- 휴가 정책 관련: 경영관리실 문의
