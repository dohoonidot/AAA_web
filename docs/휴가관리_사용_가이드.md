# 휴가관리 사용 가이드

> 휴가관리 시스템의 각 기능에 대한 상세 설명

---

## 1. 휴가 신청

### 기능 설명
새로운 휴가를 신청할 수 있는 기능입니다.

### 사용 방법
1. 상단의 **"휴가 신청"** 버튼 클릭
2. 휴가 신청 폼에서 필요한 정보 입력:
   - **휴가 종류**: 연차, 반차, 특별휴가 등 선택
   - **시작일/종료일**: 휴가 기간 설정
   - **사유**: 휴가 신청 사유 입력
   - **결재자**: 승인 담당자 선택
3. 입력 완료 후 **"제출"** 버튼 클릭

### 입력 항목
| 항목 | 필수 | 설명 |
|------|-----|------|
| 휴가 종류 | O | 연차, 오전반차, 오후반차, 특별휴가 등 |
| 시작일 | O | 휴가 시작 날짜 |
| 종료일 | O | 휴가 종료 날짜 (당일인 경우 시작일과 동일) |
| 사유 | O | 휴가 신청 사유 |
| 결재자 | O | 휴가 승인 담당자 |
| 참조자 | X | 휴가 신청 참조 대상 (선택) |

## 2. 휴가 캘린더

### 기능 설명
월별 휴가 현황과 공휴일을 한눈에 확인할 수 있는 달력 기능입니다.

### 주요 기능
- **월별 휴가 조회**: 해당 월의 모든 휴가 일정 표시
- **공휴일 표시**: 법정 공휴일 및 회사 휴일 표시
- **상세 정보 보기**: 날짜 클릭 시 해당 날짜의 휴가 상세 정보 표시
- **월 이동**: 좌우 화살표로 이전/다음 월 이동

### 사용 방법
1. 휴가관리 화면에서 달력 영역 확인
2. **좌우 화살표**를 클릭하여 원하는 월로 이동
3. 특정 날짜 클릭 시 해당 날짜의 휴가 상세 정보 확인
4. **"넓게보기"** 버튼 클릭 시 전체 화면 달력 표시

### 달력 표시 범례
| 색상/표시 | 의미 |
|----------|------|
| 파란색 | 승인된 휴가 |
| 주황색 | 대기중인 휴가 |
| 빨간색 | 공휴일 |
| 회색 | 취소된 휴가 |

## 3. 휴가 관리 대장

### 기능 설명
올해 연차 발생일수, 사용일수, 잔여일수를 종합적으로 확인할 수 있습니다.

### 표시 정보
- **연차 발생일수**: 올해 부여받은 총 연차 일수
- **사용일수**: 현재까지 사용한 연차 일수
- **잔여일수**: 남은 연차 일수

### 상세 내역 조회
1. 하단 테이블에서 휴가 사용 내역 확인
2. 각 행을 클릭하면 해당 휴가의 상세 정보 표시:
   - 휴가 종류
   - 휴가 기간
   - 신청 사유
   - 결재 상태
   - 결재자 정보

### 테이블 구성
| 컬럼 | 설명 |
|------|------|
| 휴가종류 | 연차, 반차 등 휴가 유형 |
| 허용일수 | 해당 휴가 종류의 총 부여일수 |
| 월별 사용현황 | 1월~12월 각 월별 사용일수 |
| 사용일수 | 총 사용한 일수 |
| 남은일수 | 잔여 일수 |

## 4. 결재 현황

### 기능 설명
휴가 신청 건의 결재 진행 상태를 실시간으로 확인할 수 있습니다.

### 결재 상태 종류
| 상태 | 설명 | 색상 |
|------|------|------|
| 대기 | 결재자 승인 대기중 | 주황색 |
| 승인 | 결재 승인 완료 | 녹색 |
| 반려 | 결재 반려됨 | 빨간색 |
| 취소 | 휴가 신청 취소됨 | 회색 |

### 필터링 기능
- 상단의 각 상태 영역을 클릭하면 해당 상태의 휴가 목록만 필터링하여 표시
- 예: "대기" 클릭 시 대기중인 휴가만 표시

### 결재 현황 카드
```
┌─────────────────────────────────────┐
│  대기중: 2    승인됨: 15    반려됨: 1  │
└─────────────────────────────────────┘
```

## 5. 사이드바 메뉴

### 기능 설명
다양한 휴가 관련 기능에 빠르게 접근할 수 있는 사이드바 메뉴입니다.

### 메뉴 구성
| 메뉴 | 설명 |
|------|------|
| 부서 휴가 현황 | 부서원들의 휴가 일정 달력 보기 |
| 휴가 부여 내역 | 예비군/민방위, 결혼, 경조사 연차 부여 요청 내역 |
| AI 휴가 추천 | AI 기반 휴가 계획 추천 |
| 연차 촉진 안내문 | 연차 사용 촉진 관련 안내 |

### 사이드바 고정
- 사이드바 상단의 **핀 아이콘**을 클릭하면 사이드바를 고정할 수 있습니다
- 고정 시 화면 이동 시에도 사이드바가 유지됩니다

### 관리자 기능 (휴가관리자용)
휴가관리자(경영관리실)의 경우 추가로 다음 기능을 사용할 수 있습니다:
- 예비군/민방위 연차 부여 요청 처리
- 결혼/경조사 연차 부여 요청 처리
- 직원 연차 현황 조회

## 참고 사항

### 자주 묻는 질문

**Q. 휴가 신청 후 수정이 가능한가요?**
> A. 결재 대기 상태에서는 취소 후 재신청이 가능합니다. 승인된 휴가는 취소 상신을 통해 처리합니다.

**Q. 반차는 어떻게 신청하나요?**
> A. 휴가 신청 시 휴가 종류에서 "오전반차" 또는 "오후반차"를 선택합니다.

**Q. 결재자를 변경할 수 있나요?**
> A. 결재 대기 상태에서 취소 후 재신청 시 변경 가능합니다.

### 문의처
- 시스템 오류: IT 담당자
- 휴가 정책: 경영관리실
